## How to run app interface
* Creare virtual environment
  
```python -m venv .venv ```  
```source .venv/bin/activate ```

* Clone the repo

```git clone https://github.com/gokul-gopan-k/Eye-for-Blind.git```

```cd Eye-for-Blind```

* Make the script executable:
  
```chmod +x script.sh```

* Run the script:
  
```./script.sh```

* Run the app
  
```python app.py```
